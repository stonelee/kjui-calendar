这是模块的输出文件，请 require 或 seajs.use 以上 ID 来使用本文件，具体 API 请阅读文档。

calendar

基于arale-calendar，提供了日历时间功能，按照国内习惯进行了封装

配置

target element

希望变为calendar的input元素，设置后会自动生成trigger图标，原来的input设为output

其他配置请参照arale-calendar。

needTime Boolean

设为true，显示时间功能。默认只显示小时和分钟。如果设置needSecond为true，则可以显示秒。
